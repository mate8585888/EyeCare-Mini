# 👁️ EyeCare Mini | 极简护眼小挂件

**EyeCare Mini** 是一款基于 PowerShell 开发的轻量级桌面挂件，专为长时间盯着屏幕的开发者、设计师及文字工作者设计。它遵循经典的 **20-20-20 护眼法则**，通过极简的 UI 和直观的色彩状态切换，提醒你适度休息。

---

## ✨ 核心创意亮点

* **状态色彩学**：
    * **专注模式**：显示自定义颜色（如 Cyan），静默倒计时。
    * **红色警报**：20 分钟结束，挂件瞬间变为**红色**并开启 **20 秒强制倒计时**，视觉暗示极强。
* **极致轻量**：纯 PowerShell 编写，无需安装，不占任务栏空间。
* **背景镂空**：文字悬浮感 UI，不遮挡工作内容。

---

## 🚀 快速开始

1.  **保存代码**：将脚本保存为 `EyeCare.ps1`。
2.  **执行权限**：如果无法运行，请在 PowerShell 中执行 `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`。
3.  **运行**：右键点击 `EyeCare.ps1` 选择 **“使用 PowerShell 运行”**。

---

## 🛠️ 操作说明

* **移动**：鼠标左键长按数字即可全屏拖动。
* **菜单**：右键点击数字或托盘图标，可调整颜色、字号或重置计时。
* **静音设计**：通过颜色变换进行“强视觉、弱听觉”提醒，不干扰他人。
