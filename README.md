# 👁️ EyeCare Mini | 极简护眼小挂件

**EyeCare Mini** 是一款基于 PowerShell 开发的轻量级桌面挂件，专为长时间盯着屏幕的开发者、设计师及文字工作者设计。它遵循经典的 **20-20-20 护眼法则**，通过极简的 UI 和直观的色彩状态切换，提醒你适时远眺。

---

## ✨ 核心创意亮点

* **极致轻量**：纯 PowerShell 脚本编写，无需安装任何 `.exe`，不占任务栏空间，仅通过系统托盘运行。
* **状态色彩学**：
* **专注模式**：显示你自定义的舒适色（如青色或绿色），静默倒计时，不干扰工作。
* **红警提醒**：当 20 分钟结束，挂件立即变为**醒目的红色**并开启 **20 秒强制倒计时**，视觉冲击力强，提醒优先级极高。


* **无感交互**：
* **背景镂空**：采用背景色透明技术，文字仿佛悬浮在屏幕上，与你的桌面壁纸融为一体。
* **自由拖拽**：鼠标左键随时调整位置，右键即时更改颜色和字号。
* **配置记忆**：自动保存坐标、颜色和大小，下次启动无需重复设置。



---

## 🛠️ 功能特性

* [x] **双模式自动循环**：20分钟工作与20秒休息自动轮替。
* [x] **穿透式 UI**：无边框设计，支持置顶显示。
* [x] **静音提醒**：通过颜色变换和系统气泡通知，避免在安静环境中发出尴尬的蜂鸣声。
* [x] **任务栏净化**：隐藏主程序图标，只保留右下角系统托盘操作。
* [x] **快速重置**：遇到特殊情况可随时右键点击“重置计时”。

---

## 🚀 快速开始

1. **保存代码**：将脚本保存为 `EyeCare.ps1`。
2. **运行脚本**：
* 右键点击文件，选择 **“使用 PowerShell 运行”**。
* 或者在终端执行：`powershell -ExecutionPolicy Bypass -File .\EyeCare.ps1`


3. **自定义配置**：
* **移动**：左键长按数字拖动到屏幕任意位置。
* **管理**：右键点击数字或托盘图标，可修改颜色、字号或退出。



---

## ⚙️ 配置文件 (`config.ini`)

程序启动后会在同级目录下生成配置文件，你可以手动修改或通过 UI 自动保存：

```ini
Color=Cyan       # 专注模式下的文字颜色
FontSize=18      # 字体大小
X=1680           # 窗口水平坐标
Y=950            # 窗口垂直坐标

```

---

## 💡 为什么选择这种提醒逻辑？

传统的弹窗提醒（MessageBox）往往会中断你的输入流，甚至导致全屏游戏/演示崩溃。**EyeCare Mini** 的创意在于“**视觉强暗示**”：

> 当你专注工作时，眼角余光扫到挂件变为**红色**，大脑会迅速接收到“该休息了”的信号，而不需要你手动关闭任何弹窗。
